code-reviewerを使って、コードをレビューしてください。

## 対象
$ARGUMENTS

## 作業内容
1. 対象コード・変更差分の確認
2. レビュー観点の選択
3. レビュー実施
4. レビューコメント作成
5. 重要度別に整理・報告

## レビュー観点
- コード品質
- セキュリティ（OWASP Top 10）
- パフォーマンス
- ロジック

---

## 使用例

```
/agent-dev-review このコードをレビューして
/agent-dev-review PRをレビューして
/agent-dev-review セキュリティ観点でチェックして
```
