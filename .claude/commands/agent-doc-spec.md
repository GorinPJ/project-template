spec-advisorを使って、要件定義・仕様固めを支援してください。

## 要件
$ARGUMENTS

## 作業内容
- クライアント要件のヒアリング支援
- 仕様の曖昧点の洗い出し
- 質問リストの作成
- 機能要件・非機能要件の整理
- 仕様書ドラフト作成
- 仕様レビュー・指摘
- 仕様変更の影響分析
- Confluenceへの仕様ドキュメント反映

## 出力先
- 機能要件 → 02_要件定義/機能要件
- 非機能要件 → 02_要件定義/非機能要件
- 画面一覧 → 02_要件定義/画面一覧
- 提供資料 → 02_要件定義/資料提供/提供資料

---

## 使用例

```
/agent-doc-spec この要件について仕様を固めたい
/agent-doc-spec 仕様書をレビューして
/agent-doc-spec クライアントへの質問リストを作成して
/agent-doc-spec この変更の影響範囲を分析して
/agent-doc-spec 機能仕様書のドラフトを作成して
```
