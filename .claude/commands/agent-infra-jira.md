jira-managerを使って、Jiraチケットを管理してください。

## 要件
$ARGUMENTS

## 作業内容
- チケットの作成
- チケットの更新（ステータス、担当者、コメント）
- チケットの検索
- スプリント管理
- 作業ログの記録

## 使用MCP
- mcp-atlassian

---

## 使用例

```
/agent-infra-jira Jiraチケットを作成して
/agent-infra-jira このタスクのステータスを更新して
/agent-infra-jira 現在のスプリントのチケット一覧を見せて
/agent-infra-jira このバグをJiraに登録して
```
