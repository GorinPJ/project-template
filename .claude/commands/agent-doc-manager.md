doc-managerを使って、ドキュメントを作成・管理してください。

## 要件
$ARGUMENTS

## 作業内容
- 仕様書の作成（Excel/Markdown）
- README/API仕様書の生成
- MDファイルの配置・整理
- 重複チェック・整合性確認
- index.mdの更新

## 配置ルール
- ルール系: `.claude/docs/rules/`
- ガイド系: `.claude/docs/guides/`
- カタログ系: `.claude/docs/catalogs/`

---

## 使用例

```
/agent-doc-manager この機能の仕様書を作成して
/agent-doc-manager READMEを作成して
/agent-doc-manager ドキュメントを整理して
```
