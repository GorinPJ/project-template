refactoring-advisorを使って、コードをリファクタリングしてください。

## 対象
$ARGUMENTS

## 作業内容
1. 対象コードの分析
2. 問題点・改善ポイントの特定
3. 改善案の提示（承認待ち）
4. 段階的リファクタリング実施
5. 検証・動作確認

## 注意事項
- 小さな変更を積み重ねる
- テストがあれば実行して確認
- リグレッションに注意

---

## 使用例

```
/agent-dev-refactor このコードをリファクタリングして
/agent-dev-refactor 重複コードを整理したい
/agent-dev-refactor このクラスを分割したい
```
