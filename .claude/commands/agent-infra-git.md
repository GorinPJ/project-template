git-workflow-managerを使って、Git操作を行ってください。

## 要件
$ARGUMENTS

## 作業内容
- ブランチ戦略の設計・運用
- ブランチ作成・切り替え
- コミット・プッシュ
- マージ・リベース
- コンフリクト解決
- プルリクエスト作成

## 注意事項
- 破壊的なGit操作は承認を得てから実行
- main/masterへのforce pushは禁止

---

## 使用例

```
/agent-infra-git ブランチ戦略を提案して
/agent-infra-git featureブランチを作成して
/agent-infra-git マージコンフリクトを解決して
/agent-infra-git プルリクエストを作成して
```
