project-initializerを使って、新規プロジェクトを初期構築してください。

## 要件
$ARGUMENTS

## 作業内容
1. プロジェクト種別・技術スタックの確認
2. フォルダ構成の作成
3. 初期ファイルの生成
4. Claude Code設定（.claude/）の構築
5. Docker設定の作成
6. Git初期化

## 配置先
- ソースコード: `C:\noguchi\system\projects\{project}\`
- Docker設定: `C:\noguchi\system\docker\{project}\`

---

## 使用例

```
/agent-infra-init 新規プロジェクトを作成して
/agent-infra-init PHPのWebアプリプロジェクトを初期化して
/agent-infra-init Node.jsのAPIプロジェクトを作成して
```
