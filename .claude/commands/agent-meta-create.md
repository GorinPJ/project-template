agent-creatorを使って、新しいサブエージェント・カスタムコマンドを作成してください。

## 要件
$ARGUMENTS

## 作業内容
1. 要件をヒアリングして明確化
2. 既存エージェントを参考に設計
3. サブエージェント定義ファイルを作成
4. カスタムコマンドを作成
5. カタログに登録

## 参照ファイル
- 作成ガイド: `.claude/docs/guides/agent-command-guide.md`
- 既存エージェント: `.claude/agents/`
- カタログ: `.claude/docs/catalogs/agent-catalog.md`

---

## 使用例

```
/agent-meta-create コードレビュー用エージェント
```
