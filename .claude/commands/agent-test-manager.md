test-managerを使って、テストを管理・実行してください。

## 要件
$ARGUMENTS

## 作業内容
- テスト仕様書の作成
- E2Eテストの実装
- テストの実行
- テスト結果の生成（Excel/レポート）
- カバレッジ確認

## 出力先
- テストコード: `tests/e2e/`
- テスト結果: `work/テスト結果/`

---

## 使用例

```
/agent-test-manager この機能のテスト仕様書を作成して
/agent-test-manager E2Eテストを作成して
/agent-test-manager テストを実行して
/agent-test-manager テスト結果をExcelにまとめて
```
