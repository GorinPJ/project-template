data-managerを使って、Excel・スプレッドシートを操作してください。

## 要件
$ARGUMENTS

## 作業内容
- Excelファイルの作成・読み取り・更新
- Googleスプレッドシートの操作
- データ変換（Excel ↔ スプレッドシート）
- レポート生成

## 使用MCP
- excel-mcp-server
- mcp-google-sheets

---

## 使用例

```
/agent-doc-data このデータをExcelにまとめて
/agent-doc-data Excelファイルを読み込んで
/agent-doc-data スプレッドシートを更新して
```
