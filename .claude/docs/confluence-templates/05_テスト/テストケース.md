### 管理ツール

| 用途 | ツール |
|------|--------|
| テストケース管理 | TestRail / Excel |


## テスト結果記録

### 結果ステータス

| ステータス | 意味 |
|-----------|------|
| Pass | テスト成功 |
| Fail | テスト失敗（不具合） |
| Block | テスト実行不可（環境問題等） |
| Skip | スキップ（対象外） |
