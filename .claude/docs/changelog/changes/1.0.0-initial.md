# v1.0.0 初期リリース詳細

リリース日: 2026-01-10

## 概要

プロジェクトテンプレートの初期リリース。
Claude Codeを活用したソフトウェア開発プロジェクトの標準構成を提供。

## 設計方針

### エージェント統合

当初24個のエージェントを計画したが、役割の重複を排除し16個に統合。

**統合例**:
- pattern-developer → code-developer に統合
- test-spec-generator, test-result-generator, e2e-test-generator → test-manager に統合

### スキルとエージェントの分離

- **スキル**: 自動適用されるルール・ガイドライン
- **エージェント**: 明示的に呼び出すタスク実行者

この分離により、ルールの一貫した適用と柔軟なタスク実行を両立。

### テンプレート同期機能

template-managerエージェントによるプロジェクト間同期を実装。
- project-registry.mdでプロジェクトを管理
- CHANGELOGでバージョン追跡
- 差分マージによる更新
