## 登録方法

新しいプロジェクトにテンプレートを適用した場合、このファイルに追記してください。

### 必要項目
- **プロジェクト名**: 識別しやすい名称
- **パス**: プロジェクトの絶対パス
- **テンプレートバージョン**: 適用時のバージョン（CHANGELOG参照）
- **最終同期日**: 最後にテンプレートと同期した日付
- **備考**: プロジェクトの簡単な説明

## 同期管理

### テンプレート更新時の同期フロー

1. テンプレートを更新
2. CHANGELOGに変更内容を記録
3. `/agent-meta-template` で各プロジェクトに同期

```
/agent-meta-template テンプレートの更新内容を全プロジェクトに同期してください
```

### 同期確認コマンド

```
/agent-meta-template 登録プロジェクトのテンプレートバージョンを確認してください
```

### 同期対象ファイル

以下のファイルが同期対象となります：

```
.claude/
├── agents/           # 全ファイル同期
├── commands/         # 全ファイル同期
├── skills/           # 全ファイル同期
├── mcp/templates/    # 全ファイル同期
└── docs/             # catalogs/project-registry.md以外を同期
```

**注意**: プロジェクト固有の設定ファイルは同期対象外です。
- `.claude/mcp.json` - プロジェクト固有
- `.claude/settings.local.json` - プロジェクト固有
- `CLAUDE.md` - プロジェクト固有

## バージョン管理

### セマンティックバージョニング

- **メジャー (X.0.0)**: 破壊的変更（既存プロジェクトへの影響あり）
- **マイナー (0.X.0)**: 機能追加（後方互換性あり）
- **パッチ (0.0.X)**: バグ修正・軽微な変更

### 現在のテンプレートバージョン

**1.0.0** (2026-01-10)

詳細は [CHANGELOG](../changelog/CHANGELOG.md) を参照。

## トラブルシューティング

### 同期エラーが発生した場合

1. プロジェクトパスが正しいか確認
2. ファイルのアクセス権限を確認
3. プロジェクト固有のカスタマイズがないか確認

### バージョン不整合の解決

```
/agent-meta-template pressanceプロジェクトを最新テンプレートに更新してください
```

差分を確認し、マージ方法を選択できます。
